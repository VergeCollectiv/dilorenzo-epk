# DILORENZO EPK — Photo Guide

Drop your photos here with these exact filenames. Compress each to under 500KB before
adding (aim for 1400px on the longest side at ~78% quality).

Compress command (Mac):
  sips -Z 1400 -s formatOptions 78 INPUT.jpg --out OUTPUT.jpg

## Placeholder behavior

If a photo file is missing, the section shows a dark gradient background with a faint
label. The layout stays intact — nothing breaks. Add files one at a time and refresh.
